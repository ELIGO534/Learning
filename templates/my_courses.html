    CSS Box Model Reference

    The CSS Box Model

    Box Model Components

    Content
    • Content - The actual content of the box
    • Padding - Clears an area around the content
    • Border - A border that goes around the padding
    • Margin - Clears an area outside the border

    Box Model Properties

    .box {
  width: 300px;
  height: 200px;
  padding: 20px;
  border: 5px solid #333;
  margin: 30px;
}

    Total width = width + padding-left + padding-right + border-left + border-right

    Total height = height + padding-top + padding-bottom + border-top + border-bottom

    Question: If an element has width: 200px, padding: 10px, and border: 2px, what is its total width?

    Advanced Box Model Techniques

    Box Sizing

    /* Default box model */
.default-box {
  box-sizing: content-box; /* default */
  width: 300px;
  padding: 20px;
  /* Total width = 300 + 20 + 20 = 340px */
}

/* Border-box model */
.better-box {
  box-sizing: border-box;
  width: 300px;
  padding: 20px;
  /* Total width = 300px (includes padding) */
}

    Using box-sizing: border-box makes layouts much easier to calculate.

    Margin Collapsing

    When two vertical margins meet, they collapse to the larger of the two:

    Element A (margin-bottom: 20px)
    Element B (margin-top: 30px)

    Actual gap between them: 30px (not 50px)

    Colors
    bg-*, text-*, border-*
    Display
